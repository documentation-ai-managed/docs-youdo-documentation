## Release History

<Update label="2024-10-20" description="v2.1.0" tags={["feature","improvement"]}>

## New Features

- Introduced event hosting with live chat integration for virtual readings.
- Added translation tools to assist multilingual content creation.

## Improvements

- Enhanced mobile responsiveness for better touch interactions.
- Optimized search algorithms to surface relevant literary discussions faster.

## Bug Fixes

- Resolved notification delivery delays during peak usage.
- Fixed profile image upload issues on certain browsers.

</Update>

<Update label="2024-09-15" description="v2.0.0" tags={["feature","breaking"]}>

## New Features

- Launched community forums with threaded replies for deeper conversations.
- Integrated direct messaging for private collaborations among creators.

## Breaking Changes

- Updated authentication flow to require two-factor for admin actions.
- Deprecated legacy markdown parser in favor of a more robust engine.

## Improvements

- Streamlined dashboard layout for quicker access to writing tools.

</Update>

<Update label="2024-08-05" description="v1.0.0" tags={["feature"]}>

- Core platform launch with post creation, commenting, and user profiles.

- Community guidelines and moderation tools established.
